# Creativity Generation Unit (CGU)

> 🎨 **創意點子產生器** - 一個基於 MCP 的 Agent-to-Agent 創意發想服務

---

## 💡 核心洞察

### 創意的本質

> **"All models are wrong, but some are useful!"**
> — George Box

**關鍵發現：創意可以從局部資訊產生！**

- 人類**不需要**知道整個 World Model 就能產生創意點子
- 小學生可以有創意，成人也可以，不同領域都能產生創意
- 這意味著：**創意不需要完整知識，只需要足夠的連結能力**

### 為什麼用 MCP + A2A？

| 優勢 | 說明 |
|------|------|
| **模型無關性** | 即使最簡單的模型也能提供其具備的創意發想能力 |
| **可組合性** | 多個 Agent 可以協作，碰撞出更多火花 |
| **專注職責** | CGU 專注於創意生成，主 Agent 專注於任務執行 |

### 🔷 2. 心智圖 (Mind Map)

Tony Buzan 發明的放射性思考工具：

```
                    ┌─ 子主題 1.1
            ┌─ 主題 1 ─┼─ 子主題 1.2
            │       └─ 子主題 1.3
            │
            │       ┌─ 子主題 2.1
中心主題 ───┼─ 主題 2 ─┼─ 子主題 2.2
            │       └─ 子主題 2.3
            │
            │       ┌─ 子主題 3.1
            └─ 主題 3 ─┼─ 子主題 3.2
                    └─ 子主題 3.3
```

**Agent 實現：**
```python
def mind_map_expansion(center_concept, depth=3, branches=5):
    """
    從中心概念向外放射擴展
    每層產生 branches 個分支
    遞迴到 depth 層
    """
    map = {center_concept: []}
    for _ in range(branches):
        branch = generate_related_concept(center_concept)
        map[center_concept].append(branch)
        if depth > 1:
            map.update(mind_map_expansion(branch, depth-1, branches-1))
    return map
```

### 🔷 10. 形態分析法 (Morphological Analysis)

Fritz Zwicky 發明的組合創新法：

```
維度        選項 1      選項 2      選項 3      選項 4
─────────────────────────────────────────────────────
動力來源    電力        汽油        太陽能      人力
輪子數量    2           3           4           無
材質        金屬        塑膠        木頭        碳纖維
用途        通勤        運動        載貨        表演

組合數 = 4 × 4 × 4 × 4 = 256 種可能
```

**Agent 實現：**
```python
def morphological_matrix(dimensions: dict) -> list:
    """
    dimensions = {
        "動力": ["電力", "汽油", "太陽能"],
        "輪數": [2, 3, 4],
        "材質": ["金屬", "塑膠", "碳纖維"],
    }
    回傳所有組合
    """
    from itertools import product
    return list(product(*dimensions.values()))
```

## 🔥 火花碰撞機制

### 低關聯但有潛力的連結

```python
# 概念示意
def find_creative_connections(concept, creativity_level):
    """
    尋找有潛力碰出火花的概念
    
    關鍵：不是亂數，而是有某種關聯但關聯性很低
    """
    
    # 獲取概念的多維度屬性
    attributes = extract_attributes(concept)
    # 結構、功能、情感、歷史、社會...
    
    # 根據創意層級決定搜尋範圍
    search_radius = map_level_to_radius(creativity_level)
    
    # 尋找在某個維度上有微弱關聯的概念
    candidates = []
    for attr in attributes:
        weak_links = find_weak_associations(attr, radius=search_radius)
        candidates.extend(weak_links)
    
    # 評估碰撞潛力（不是隨機，是有依據的）
    return rank_by_spark_potential(candidates)
```

### 為什麼低關聯能產生創意？

1. **突破思維定勢** - 高關聯會導向常見答案
2. **新組合空間** - 低關聯開啟未探索的組合
3. **驚喜元素** - 意想不到的連結產生 "Aha!" 時刻
4. **類比遷移** - 遠距離類比往往最有洞察力

      constraints?: string[];
    };
    output: {
      ideas: Idea[];
      connections: Connection[];
    };
  };
  
  // 概念碰撞
  sparkCollision: {
    input: {
      conceptA: string;
      conceptB: string;
    };
    output: {
      sparks: string[];
      rationale: string;
    };
  };
  
  // 聯想擴展
  associativeExpansion: {
    input: {
      seed: string;
      direction: 'similar' | 'opposite' | 'random' | 'cross-domain';
      depth: number;
    };
    output: {
      associations: Association[];
    };
  };
}

## 🌟 設計原則

### 1. 模型民主化
> 即使最簡單的模型也有其獨特的創意視角

### 2. 局部即足夠
> 不需要完整世界模型，局部資訊就能產生有價值的創意

### 3. 連結重於知識
> 創意的本質是連結，不是資訊量

### 4. 錯誤也有用
> All models are wrong, but some are useful
> 「錯誤」的連結可能正是創新的來源
